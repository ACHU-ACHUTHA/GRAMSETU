# GRAMSETU 🌾🏥  
**Offline-First Community Health Web App**

## Overview
GramSetu is a **Offline-first web application** built to support community healthcare in rural areas with limited or no internet access.  
The site enables patient registration, health screening, risk assessment, and emergency referrals using **local data storage, SMS, and QR codes**.
---
## Key Features
- React-based frontend (Vite)  
- Patient registration and record management  
- Health screening with risk classification  
- Offline data storage  
- High-risk referrals via SMS & QR codes  
---
## Tech Stack
- **Frontend:** React  
- **Storage:** LocalStorage / IndexedDB  
- **Data Transfer:** SMS, QR Codes  
---
> Note: React uses a single HTML entry file. All UI logic is inside `src`.
